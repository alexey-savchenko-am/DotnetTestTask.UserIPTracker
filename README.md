# DotnetTestTask.UserIPTracker
## Setup

```bash
git clone https://github.com/alexey-savchenko-am/DotnetTestTask.UserIPTracker.git
cd DotnetTestTask.UserIPTracker
docker compose up --build --force-recreate
```
Ensure that ports **8080**, **8081**, **5432** are available on your machine. If not, update them in **docker-compose.yml**.

**Swagger for the backend API**: [http://localhost:8080/swagger/index.html](http://localhost:8080/swagger/index.html)
